📄 AI Resume Analyzer using NLP
-

An intelligent AI-powered Resume Analyzer built using Python, NLP, and Streamlit that extracts information from resumes (PDF/DOCX), analyzes skills, and provides insights to improve resume quality.

🚀 Live App:https://kisxlvdgvqmvq7kvrmuhog.streamlit.app/

📦 Tech Stack: Python | Streamlit | spaCy | NLP | Machine Learning


🔍 Features
-

📑 Upload resumes in PDF or DOCX format

🧠 Extract text using NLP techniques

🏷️ Identify key skills and keywords

📊 Analyze resume content against job requirements

💡 Provide improvement tips and feedback

⚡ Fast, interactive web interface using Streamlit


🛠️ Technologies Used
-

Python
Streamlit – Web UI

spaCy – Natural Language Processing

pdfplumber – PDF text extraction

docx2txt – DOCX parsing

scikit-learn – ML utilities

🎯 Use Cases
-

Students improving resumes for placements

Freshers preparing for internships

Job seekers optimizing resumes

Resume screening assistance

NLTK – Text preprocessing

Pandas & NumPy – Data handling
